# URL Encoder/Decoder + Base64 Converter

This tool allows you to:

- **Encode/Decode URLs**: Convert special characters in URLs to their percent-encoded equivalents and vice-versa.
- **Convert Text to/from Base64**: Encode any text into Base64 format and decode Base64 strings back to text.
- **URL-safe Option**: Provides an option for URL-safe Base64 encoding/decoding.

## How to Use:

1. **URL Encoding/Decoding**:
   - Enter the URL or text you want to encode/decode in the input field.
   - Select the appropriate action (Encode URL / Decode URL).
   - The result will be displayed in the output field.

2. **Base64 Conversion**:
   - Enter the text or Base64 string you want to convert in the input field.
   - Select the appropriate action (Encode Base64 / Decode Base64).
   - Check the 'URL Safe' option if you need to use URL-safe Base64.
   - The result will be displayed in the output field.

This tool is useful for developers, network administrators, and anyone who needs to manipulate URL parameters or handle Base64 encoded data.
